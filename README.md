🖐️ Smart Hand Mouse Control System

A Python-based AI hand gesture control system that allows users to control the mouse using hand gestures through a webcam.
Built using:

* OpenCV
* MediaPipe
* PyAutoGUI

🚀 Features

✅ Real-time hand tracking
✅ Mouse cursor movement using index finger
✅ Single Click gesture
✅ Double Click gesture
✅ Scroll Up / Scroll Down
✅ Screenshot capture gesture
✅ Smooth cursor movement
✅ Gesture-based interaction without physical mouse

# 📸 Gestures

| Gesture                         | Action             |
| ------------------------------- | ------------------ |
| Move Index Finger               | Move Cursor        |
| Thumb + Index Finger Close      | Single Click       |
| Double Pinch Quickly            | Double Click       |
| 3 Fingers Up                    | Enable Scroll Mode |
| Move Finger Up in Scroll Mode   | Scroll Up          |
| Move Finger Down in Scroll Mode | Scroll Down        |
| Fist (0 Fingers)                | Take Screenshot    |


🛠️ Technologies Used

* Python
* OpenCV
* MediaPipe
* PyAutoGUI
* Math
* Time

 🧠 How It Works

1. Webcam captures live video.
2. MediaPipe detects hand landmarks.
3. Finger positions are analyzed.
4. Gestures are recognized.
5. Corresponding mouse actions are executed.
